What is Pixel Suite?
Pixel Suite is an all-in-one terminal-based utility and gaming hub built entirely in Lua. It features a cool, customizable UI, persistent user profiles, a secure database, and a personalized user experience.
🔥 Features in vF1.3.2 Pixel Suite Fantastically Polished.
🔒 Secured User Database: Accounts are secured with password obfuscation to protect user data from casual snooping.
👥 Role-Based Access Control (RBAC): Distinct permissions for Admins and Users.
💾 Automatic Backups: Database files are automatically backed up before saving.
🔢 Pixel Calculator: A robust tool for basic arithmetic with input validation.
🎮 Minigames: Includes a versatile RNG generator, Ping-Pong, and a FOTD (Fact of the Day) engine.
📰 News Archive: Stay updated with the latest version changes directly in the terminal.
🛡️ Quality of Life Improvements: Strict input validation (no blank inputs), and character limits for usernames (23) and passwords (40).

🛠️ Technical Specs
Language: Lua 5.1+ (Recommended: 5.4)
Platform: Cross-platform (Android via ACode, PC/iOS via Lua Interpreter)
Database: Terminal-based text storage (pixel_db.txt)
🤖 API Integration (Advanced)
Pixel Suite has the capability to integrate with external AI APIs.
To configure API functionality:
Obtain an API Key from your service provider.
Implement the socket.http library within the Lua environment.
Store the API Key securely within the script variables.
💾 Installation
Ensure you have a Lua interpreter installed, it should absolutely need to support standard io libraries (io.read(),io.open), and ANSI escape codes for colored texts.
Download PixelSuite.lua.
Run the script in your terminal: lua PixelSuite.lua, commands should be easy as buttons are assigned to a number you'll need to enter in the terminal.
