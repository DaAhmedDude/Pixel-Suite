What is Pixel Suite?
Pixel Suite is an all-in-one terminal-based utility and gaming hub built entirely in Lua. There is a cool, customizable UI and a personalized user experience.
This beast features:
🔢 Pixel Calculator: A robust tool for basic arithmetic with error-handling.
🎮 Minigames: Includes "Shi-Fu-Mi" (Rock-Paper-Scissors) with a random-logic engine and a versatile RNG.
🎨 Dynamic Themes: Change the look of the entire suite on the fly with built-in color profiles.
⚙️ System Identity: Personalized boot sequence and user profile tracking.
Hidden Secrets:
Try to find the "Magic Number" or the "Magic word" if you can.
Technical Specs:
Language: Lua 5.4
Platform: Cross-platform (Android/PC/iOS via Lua Interpreter)
